# odin-recipes
Build a basic recipe website

Using command line, git, and HTML basics to create a website with three recipes.
The website should have clear directory , where each folder contains the webpages or images.
The website should have a clear head and body elements.
The body element should contain an image, clearly labelled, and also links to the three recipes in recipe directory.
On each recipe, there should be name of the dish, picture, ingredients (unordered list) and steps (ordered list).
